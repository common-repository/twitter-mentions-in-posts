Show tweets about your posts right under them.

== Description ==

This plugin fetch all tweets from Twitter, which has been written about some of your posts. For new article are changes cheching every hour. For oldone it is question one or two days.

You can set how many tweets to show and in swich format. It can be regular HTML or embed tweets generated by Twitter javascript.

List of tweets is automaticky show above comment form, but your can also do manually by put funciont `<?php tmip_show_tweets() ?>` into your WordPress theme.

Just remember, Twitter allow to search tweets only few days old, so i will show probably only in new articles. Be patient.

== Installation ==


1. Upload `twitter-mentions-in-posts` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to setting page where you can set automatic showing above comment form or decide to place function `<?php tmip_show_tweets() ?>` manually
